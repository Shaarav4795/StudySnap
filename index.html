How to Install

Get started with StudySnap on your iOS device.

1

Clone the Repository

Clone the StudySnap repository from GitHub to your local machine.

git clone https://github.com/Shaarav4795/StudySnap.git
2

Open in Xcode

Open the StudySnap.xcodeproj file in Xcode.

3

Build & Run

Select your device or simulator and press the Run button (Cmd + R).

4

Choose Your Model

Select your preferred AI model in the app settings to suit your needs.
